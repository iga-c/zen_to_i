# ZenToI
兆までの漢数字を半角数字に変換します。
漢数字が続いていたらそれぞれ変換します。

[![Build Status](https://travis-ci.org/yoshitsugu/zen_to_i.svg)](https://travis-ci.org/yoshitsugu/zen_to_i)

## Installation

Add this line to your application's Gemfile:

    gem 'zen_to_i'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zen_to_i

## Usage

```ruby
"hoge".zen_to_i
#=> "hoge"

"一二三".zen_to_i
#=> "123"

"百三".zen_to_i
#=> "103"

"三兆五十二万四十八".zen_to_i
#=> "3000000520048"
```
